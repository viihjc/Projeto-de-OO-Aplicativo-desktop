# Projeto Sistema Bancário em Python - Aplicativo-desktop

Este é um sistema bancário desenvolvido em Python que integra Programação Orientada a Objetos com persistência em banco de dados relacional (SQLite).

## Objetivo
Gerenciar clientes, contas e transações de forma estruturada e segura.

## Tecnologias
- Banco de Dados: SQLite (`banco.db`)
- ORM: SQLAlchemy
